
 A modern plant e-commerce storefront built with Next.js, TypeScript, and Tailwind CSS.
FloraVision is a responsive, dark-themed plant shopping UI showcasing curated indoor plants with sections for top-selling items, trending picks, O2-producing plants, and customer reviews.
IT FOCUSES ON:
Responsive landing page
Reusable React components
Plant product cards
Trending and top-selling section

TECH STACK: 
Next.js
Typescript
Tailwind CSS
